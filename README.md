### Run as dev locally:

- `npm i`
- create and fill `.env` file as shown in `.env.example`
- `npx prisma migrate dev`
- `npm run dev`

### Run as dev with Docker:

- `npm i`
- `docker-compose up`

### Run as prod locally:

- `npm i`
- `npm run build`
- create and fill `.env.prod` file as shown in `.env.example`
- `npm start`

### Run test :

- `npm i`
- `npm run tesT`

### Run test as dev:

- `npm i`
- `npm run test:dev`
