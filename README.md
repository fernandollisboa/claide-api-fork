### Run as dev:

- `npm i`
- create and fill `.env` file as shown in `.env.example`
- `npx prisma migrate dev`
- `npm run dev`

### Run as prod:

- `npm i`
- `npm run build`
- create and fill `.env` file as shown in `.env.example`
- `npm start`

#### generate and run migrations

1. npx prisma migrate dev --name "example"
