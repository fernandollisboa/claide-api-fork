### Run as dev locally:

- `npm i`
- create and fill `.env` file as shown in `.env.example`
- `npx prisma migrate dev`
- `npm run dev`

## Run as dev with Docker:
- npm i

### Run as prod locally:

- `npm i`
- `npm run build`
- create and fill `.env.prod` file as shown in `.env.example`
- `npm start`

### Run tests:

- `npm i`
- `npm run test`
